#  Moduler & Packager

##  Project Overview
**Multi-Utility Toolkit** ek menu-driven Python application hai jo multiple utilities ek hi program me provide karta hai.  
Is project ka purpose Python ke **built-in modules**, **functions**, **loops**, **match-case**, aur **file handling** concepts ko practically demonstrate karna hai.

##  Features
This toolkit provides the following utilities:

###  Date & Time Operations
- Display current date and time
- Calculate difference between two dates
- Stopwatch
- Countdown timer

###  Mathematical Operations
- Factorial calculation
- Compound interest calculation
- Trigonometric calculations (sin, cos, tan)
- Area of circle

###  Random Data Generation
- Generate random number
- Generate random list
- Create random password
- Generate OTP

###  Unique Identifier
- Generate UUID (Universally Unique Identifier)

###  File Operations
- Create a new file
- Write to a file
- Read from a file
- Append data to a file

###  Explore Python Modules
- View attributes of any Python module using `dir()`


##  Technologies & Modules Used
- `math`
- `random`
- `uuid`
- `datetime`
- `time`
